## 任务2：
编写程序实现：
①定义一个大小为 10 的整形列表 a；
②从键盘输入 10 个整数，放置到列表 a 中；
③输出列表 a 中的最大值。
要求：使用列表、循环结构语句实现。
1.第一种方法
```python
print("定义一个大小为10的列表a")
a = list(range(0,10))
print("输入10个整数，放入a列表中")
for i in range(0,10):
    a[i] = int(input(f"第{i+1}次输入："))
print(a)
print("输出a列表中最大值")
max = a[0]
for i in range(1,10):
    if a[i]>max:
        max=a[i]
print(f"最大值为{max}")
```
## 任务3：
请编写函数(或方法)fun，其功能是：计算正整数 n 的各位上的数字之积，将结果放到 c 中。
1.第一种方法
```python
def fun(N):
    G = 1
    for i in range(0,len(N)):
        G *= int(N[i])
    return G
N = input("请输入一个任意的正整数")
V = fun(N)
print(f"正整数为{N}的各位上的数字之积为：{V}")
```
